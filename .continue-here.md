<decisions_made>

- Recording window: separate frameless BrowserWindow (alwaysOnTop, skipTaskbar, movable, focusable) — NOT inline modal
- System audio: desktopCapturer (built-in Electron, least user friction) — macOS shows screen recording permission dialog once
- Session schema: audioSources: AudioSource[] array replaces single audioFile field, with in-place migration
- Single session created after recording (not two), with two pre-assigned speakers "You" (mic) and "Other" (system audio)
- Files written incrementally via IPC chunks (timeslice: 1000ms) — data not lost on crash
- Transcription: "Transcribe All Sources" button for recorded sessions — runs whisper on each source, merges segments by timestamp
- Speakers auto-upserted: "You" and "Other" (reuse if already exist)
- Permissions: PermissionsGate shown inside recording window if mic not granted; screen recording optional (system audio degrades gracefully)
  </decisions_made>
