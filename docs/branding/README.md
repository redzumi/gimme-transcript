# Branding Assets

SVG assets for repository presentation and launch posts.

## Files

- `repo-social-card.svg` — `1280x640`, for GitHub social preview and link cards
- `repo-banner.svg` — `1440x520`, for README or landing sections
- `post-square.svg` — `1080x1080`, for X, Telegram, LinkedIn, and generic social posts
- `post-landscape.svg` — `1600x900`, for announcements, blog headers, and wider posts

## Notes

- All assets are plain SVG and can be exported to PNG with any vector editor or browser.
- Palette follows the app logo: warm orange gradient, dark graphite, soft cream panels.
- Typography is outlined as text in SVG, so final rendering depends on available system fonts.
